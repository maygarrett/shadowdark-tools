# Shadowdark Tools

Local-first character tools for a Star Wars Shadowdark campaign.

Hosted app:

https://maygarrett.github.io/shadowdark-tools/

## Player Data

Characters are saved locally in each player's browser. There is no backend,
authentication, or sync service.

Players can share characters with the GM by opening a saved character and using
Export to download a character JSON file. The GM can use Import Character on the
character library page to load that JSON file into their own browser.

Clearing browser storage, using a private browsing session, or switching to a
different browser or device can remove or hide saved characters. Export important
characters before clearing browser data.
